# UI/UX & Design System Guidelines

## Core Principles
1. **Aesthetics:** The design must be premium, immersive, and evoke a sense of nature and relaxation.
2. **Colors:** Use earthy tones (greens, browns, wood colors) combined with clean whites and modern dark modes. Avoid generic primary colors (pure red/blue). Use harmonious palettes.
3. **Typography:** Use modern web fonts (e.g., Inter, Montserrat, or Outfit) instead of default browser fonts.
4. **Components:** High-quality imagery, glassmorphism where appropriate, rounded corners for cards and buttons.
5. **Interactions:** Subtle micro-animations on hover and active states to make the interface feel alive. Avoid rigid, static elements.

## Styling Approach
- Use Vanilla CSS for core styles.
- Avoid inline styles.
- Maintain responsive design principles (Mobile First).

## Reference
Always ensure the UI "WOWs" the user. Do not build minimal viable prototypes when creating new components.
